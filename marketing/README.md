# Ape Adventure — Marketing & Brand

Strategic playbook for taking *Ape Adventure* — a 90s-style jungle platformer — from "scaffold" to "shipped." This folder is the single source of truth for who the game is, who it's for, and how we tell people about it.

## Folder map

| Folder | What lives here |
|---|---|
| `brand/` | Identity, voice, and visuals — who we are |
| `strategy/` | Positioning, audience, competitors — where we play |
| `launch/` | Vertical-slice launch plan + press kit — how we ship |
| `channels/` | Social and community playbooks — where we show up |

## How to use this

- Read `brand/identity.md` first — it anchors everything else.
- The vertical slice is the **first marketing milestone**. Don't promote a vapor full game; promote what's playable.
- Update files when reality shifts. Strategy docs that go stale lie to you.
- Keep one idea per file. If a file grows past ~150 lines, split it.

## What changed from v1

The previous brand framed *Ape Escape* as a 1981-arcade homage with the constraint "single file, no assets, vector look." That game still exists as the parked legacy v0 (`ape_escape.py`). The new project, **Ape Adventure**, is a 90s-console-platformer homage with the opposite aesthetic: rich sprites, painted parallax, real music, multi-world. Everything in this folder is rewritten for that direction.
